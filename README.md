# Emacs-My-Ninja-Path

## Command and shortcuts

### Working with frames 
To open a new frame press: **C-x 5 f** 

To open a file in a new window**C-x 4 f** for open a file and make a frame with it. 

If you only want to replicate the currente buffer use
```make-frame``` command.
